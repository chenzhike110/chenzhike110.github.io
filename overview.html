Generative model and discriminative model

Generative models estimate the probability of observing observation x and describes how a dataset is generated. By sampling from this model, we can generate new data. They must be probabilistic rather than deterministic and include a stochastic(random) element the influences the individual samples. While the discriminative models are learning a function that maps an input to an output using labeled dataset. They estimates the probability of a label y given observation x.

As we fomulate Generative Models in a mathematic way. Suppose we are given a dataset \( \{x_1, x_2, ..., x_N \}\), where each point is drawn independently from an data distribution \( p(x) \).The goal of generative modeling is to fit a model to the data distribution such that we can synthesize new data point at will by sampling.

Type of Generative models

  1. Generative adversarial networks: GAN make unsupervised data generation task as a supervised one. The discriminator model learns to distinguish the real data from the fake samples that produced by the generative model. And two models are trained together as a minimax game.
  2. Variation autoencoders: VAE inexplicitly optimize the log-likelihood by maximizing the evidence lower bound(ELBO)
  3. Flow-based generative models: A Flow-based generative model can be viewed as invertible transformations, which explicitly learns the data distribution and trained by optimizing negative log-likelihoot.
  4. Diffusion models: Diffusion models are inspired by non-equilibrium thermodynamics, which learn to reverse the diffusion process to construct desired data from noise. Diffusion models are learned with a fixed procedure and the latent varible has high dimensionality (same as the original data).
